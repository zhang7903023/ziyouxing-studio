一、什么是今日水印相机

今日水印相机（也有叫"水印相机"、"工程水印相机"的同类App）是一款给照片添加时间、地点、项目名称等水印信息的拍摄工具，广泛应用于：

  • 工程建设：监理人员拍照记录施工进度，水印自动记录时间地点
  • 物业巡检：保安/物业拍照签到，证明已到岗巡查
  • 外勤销售：拜访客户时拍照记录拜访地点和时间
  • 物流验收：快递/物流人员拍照证明已送达指定地点
  • 现场取证：各类需要留存地理位置证据的场景

今日水印相机的核心价值在于：照片上的水印包含精确的时间、地点坐标，无法手动修改，具有法律效力。正因如此，当事人如果不在现场就无法正常拍照——这就是虚拟定位的刚需所在。

二、为什么需要改定位

典型的海外华人痛点场景：

场景：你在美国加州，但公司要求你每天用今日水印相机拍摄国内某工地的施工照片，用于工程监理报告。你不可能每天飞回国拍照，但不打卡就算旷工。

解决方案是：修改iPhone或安卓手机的GPS定位，让今日水印相机读取到的是国内目标工地的坐标，这样拍出来的照片水印显示的就是正确的地点。

三、iOS 26修改步骤

iOS 26的今日水印相机直接读取系统CoreLocation的GPS数据，改定位的核心思路是覆盖系统GPS输出。

方法一：Xcode注入法（推荐）

Step 1：确认目标坐标

用百度地图打开目标工地地址，记录下坐标（经纬度）。例如北京某工地可能是：39.9149, 116.3912。

Step 2：创建GPX文件

用文本编辑器（Mac自带TextEdit或Windows记事本）创建文件：

<?xml version="1.0" encoding="UTF-8"?>
<gpx version="1.1">
  <wpt lat="39.9149" lon="116.3912">
    <name>目标工地</name>
  </wpt>
</gpx>

将坐标替换为实际目标位置，保存为 site.gpx。

Step 3：Xcode注入

Mac打开Xcode → 连接iPhone → Window → Devices → 选设备 → Debug → Simulate Location → Add GPX File → 选择site.gpx。

Step 4：验证并拍照

打开今日水印相机，进入拍照界面，确认屏幕上显示的定位位置已变为目标地点，然后拍摄即可。拍出来的照片水印会自动使用注入的GPS坐标。

方法二：免电脑自动方案

如果不想每次用电脑注入，可以联系我们获取iOS自动续命方案，一次设置后长期生效，无需重复操作。

四、安卓修改步骤

方法一：开发者选项（无需ROOT）

  1. 设置 → 关于手机 → 连续点击版本号7次开启开发者模式
  2. 设置 → 系统 → 开发者选项 → 开启"允许模拟位置"
  3. 选择"选择模拟位置信息应用" → 选择虚拟定位App
  4. 打开虚拟定位App，搜索目标地址并设置为当前位置
  5. 打开今日水印相机，验证位置后拍照

方法二：ROOT + Xposed（更稳定）

如果手机已ROOT，可以安装Xposed框架 + 虚拟定位模块，实现系统级GPS伪造，今日水印相机会读取伪造后的坐标，效果更稳定。

方法三：定制ROM（企业用户推荐）

对于有批量外勤团队的企业，可以定制专属安卓ROM，将目标GPS坐标写入系统底层，开机即用，无需任何设置，适合管理10人以上团队。

五、常见问题

Q：改定位后拍的照片会被看出来是假的吗？

不会。今日水印相机读取的是系统级GPS坐标，照片EXIF信息里的经纬度就是真实的修改后坐标，和人工实拍没有任何区别。

Q：今日水印相机的水印位置可以手动改吗？

可以部分修改文字内容（水印模板），但GPS坐标是系统自动读取的，无法在App内手动输入，必须通过虚拟定位方式修改。

Q：iOS 26.4下今日水印相机有闪退问题吗？

目前实测正常。部分较老版本的今日水印相机App可能在iOS 26上有兼容性问题，建议更新到App Store最新版本后再操作。

Q：多个工地需要切换不同位置怎么办？

可以准备多个GPX文件，每次切换工地前在Xcode里重新注入即可。或者联系我们获取App方案，在手机上直接切换位置，无需电脑。

Q：今日水印相机的照片有法律效力吗？

正规工程场景下，水印照片具有证据效力。但请注意，通过虚拟定位方式拍摄的照片在法律层面存在争议，如果是重要工程证据，建议与公司提前沟通。
