一、飞书 vs 钉钉：选哪个

飞书（字节跳动）和钉钉（阿里巴巴）是国内两大企业办公平台，各有特点：

对比项 飞书 钉钉
所属公司 字节跳动 阿里巴巴
主要用户 互联网/科技公司为主 传统企业/制造业为主
打卡方式 GPS / WiFi / 人脸 GPS / WiFi / 蓝牙 / 人脸
定位检测强度 中等 较高
iOS 26兼容性 ✅ 良好 ✅ 良好
改定位难度 ⭐⭐ 中低 ⭐⭐⭐ 中高

两家的定位检测逻辑类似，但钉钉的防护机制更复杂一些，海外华人如果用的是钉钉，建议优先用我们的定制方案。

二、两家的防作弊机制

2026年，飞书和钉钉都已经升级到第四代考勤防作弊系统，主要检测手段包括：

GPS相关检测

  • 坐标跳跃检测：短时间内GPS坐标大幅移动（如从北京瞬间跳到纽约）
  • 坐标漂移检测：GPS信号不稳定时坐标在小范围内随机抖动
  • 多App联合校验：部分企业开启了飞书+钉钉同时打卡，位置必须一致

WiFi相关检测

  • WiFi名称检测：检测是否连接公司指定WiFi（名称必须匹配）
  • WiFi MAC地址检测：部分企业还检测路由器MAC地址

设备相关检测

  • ROOT/越狱检测：检测手机是否已ROOT（安卓）或越狱（iOS）
  • VPN检测：检测是否使用VPN，部分企业会拦截VPN流量
  • IP地理校验：服务器端比对登录IP与打卡GPS坐标是否匹配
建议：在使用虚拟定位前，先了解公司开启了哪些检测选项。基础GPS改定位可以解决大多数问题，但如果公司同时开启了WiFi+IP检测，需要组合方案。

三、飞书打卡改定位步骤

iOS 26操作流程

Step 1：确认打卡规则

飞书 → 工作台 → 考勤打卡 → 查看规则。注意看"打卡方式"：

  • 只显示地点 → 纯GPS
  • 显示地点+WiFi → GPS+WiFi
  • 有人脸图标 → 包含人脸识别，需要单独评估

Step 2：获取公司GPS坐标

用百度地图打开公司地址，长按公司位置，获取经纬度坐标。

Step 3：注入GPX文件

Mac打开Xcode → 连接iPhone → Window → Devices → 选设备 → Debug → Simulate Location → 选择GPX文件。

Step 4：验证打卡位置

打开飞书 → 工作台 → 考勤打卡 → 打卡界面，确认当前位置显示为公司地址，然后点击打卡。

安卓操作流程

  1. 开启开发者选项 → 允许模拟位置
  2. 安装虚拟定位App并设置为模拟位置应用
  3. 在虚拟定位App中搜索并设置公司地址
  4. 打开飞书 → 考勤打卡 → 验证位置 → 打卡

四、钉钉打卡改定位步骤

钉钉的定位检测比飞书更严格，基础操作流程相同，但有几个额外注意点：

iOS 26操作流程

  1. 打开钉钉 → 工作台 → 考勤打卡 → 查看规则，确认打卡方式
  2. 获取公司GPS坐标（百度地图）
  3. 通过Xcode注入GPX文件
  4. 打开钉钉打卡前，先打开一个地图App确认位置已变更
  5. 确认后在钉钉内打卡

钉钉额外注意事项

WiFi打卡：部分钉钉打卡规则开启了"必须连接公司WiFi"，这种情况下光改GPS不够，还需要模拟公司WiFi环境（可用我们的WiFi模拟器设备）。
蓝牙打卡：部分公司使用钉钉蓝牙打卡（Beacon基站），这种方式无法通过GPS改定位解决，需要单独方案。

推荐方案

  • 纯GPS打卡（钉钉）：Xcode注入 GPX → 即可
  • GPS + WiFi（钉钉）：GPX注入 + WiFi模拟器设备
  • 蓝牙打卡（钉钉）：联系我们评估定制方案

五、常见问题

Q：飞书和钉钉能同时改定位吗？

可以。iOS 26系统级的GPS注入对所有App生效，飞书和钉钉会读取同一个修改后的坐标，不需要分开设置。

Q：公司用的是蓝牙打卡，GPS改定位有用吗？

没用。蓝牙打卡依靠蓝牙Beacon基站信号，与GPS无关，需要单独的蓝牙模拟设备。如果公司同时开了GPS+蓝牙，则两个都需要处理。

Q：飞书的定位比钉钉更难改吗？

恰恰相反，飞书的定位检测机制比钉钉更宽松，对虚拟定位的容错度更高。钉钉的第四代防作弊系统会检测坐标跳跃和漂移，敏感度更高。

Q：iOS 26.4对飞书/钉钉有影响吗？

目前实测两个App在iOS 26.4上均正常工作，没有闪退或功能异常。苹果没有针对这两个App做特殊限制。

Q：改定位打卡后公司会收到提醒吗？

正常情况下不会。企业管理员只能看到打卡记录，看不到定位修改痕迹。但如果定位数据与公司IP检测产生冲突（如VPN IP在美国），可能会触发系统告警。
